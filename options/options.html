Claude Mail Assistant

Community add-on — not affiliated with Anthropic

Privacy notice: When you use any AI feature, the content of the active email and your API key are sent directly from your machine to Anthropic's API (api.anthropic.com). No data passes through any third-party server. Your API key is stored locally in Thunderbird only. Full privacy policy

Get your key at console.anthropic.com. Tip: set a monthly usage limit on that page to avoid surprise charges.

Claude Mail Assistant is open source and free to use. You pay only for your own Anthropic API usage — see anthropic.com/pricing.
