Claude Mail Assistant

Privacy Policy — last updated April 2026

Claude Mail Assistant is a free, open-source Thunderbird add-on that lets you use Anthropic's Claude AI to help compose, improve, summarize, and reply to emails. This policy explains what data is sent where and how your API key is stored.

Important: this add-on is not affiliated with or endorsed by Anthropic. "Claude" is a trademark of Anthropic, PBC. This is an independent community project.

What data is sent to Anthropic

When you use any AI feature in this add-on, the following is sent directly from your Thunderbird to Anthropic's API (api.anthropic.com):

  • The content of the email you are reading or composing
  • The subject line of that email
  • Your instruction or selected quick action
  • Your Anthropic API key (in the request header)

No data passes through any intermediate server — the request goes directly from your machine to Anthropic. This add-on has no backend of its own.

Anthropic's own privacy policy governs how they handle data sent to their API: anthropic.com/legal/privacy

What is stored locally

  • Your API key — stored in Thunderbird's local extension storage (browser.storage.local)
  • Your selected model preference

This data never leaves your machine except as part of the API request described above. It is not synced, shared, or uploaded anywhere.

What this add-on does NOT do

  • Does not collect analytics or usage statistics
  • Does not send data to any server other than api.anthropic.com
  • Does not read emails in the background — only processes the currently active email when you click a button
  • Does not store email content beyond the duration of a single API call

Your API key security

Your Anthropic API key is stored locally and sent directly to Anthropic with each request. Because this is a client-side add-on, the key is not protected by a server layer. We recommend:

  • Using a dedicated API key for this add-on with a usage limit set in the Anthropic console
  • Not sharing your Thunderbird profile with untrusted users
  • Revoking the key at console.anthropic.com if you suspect it has been compromised
